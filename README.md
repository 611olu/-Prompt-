# 国际中文教学多模态资源生成 Prompt 手册
本仓库收录《国际中文教学多模态资源生成Prompt手册》第三版，面向国际中文教学中的文本、图片、视频等多模态资源生成任务，提供可复制、可调整、可校验的 Prompt 编写范式。
## 资源简介
本手册以国际中文教学目标、语言知识体系、学习者需求和资源使用场景为基础，整理了多类教学资源生成 Prompt 模板。手册强调人机协同，不以替代教师判断为目标，而是为教师和资源建设者提供可复用的生成式资源制作方案。
手册覆盖文本类、图片类和视频类资源。文本类资源包括教学例句、共同价值标准例句等，可用于词汇释义、语法展示和语用说明；图片类资源包括词汇释义图、场景图等，可辅助学习者直观理解词义和语境；视频类资源包括语法教学视频、数字人教学视频等，可用于呈现动态过程、语法差异和文化内容。
## 主要内容

- 共同价值标准例句生成 Prompt

- 数字人教学视频生成 Prompt

## 基本原则
本手册的 Prompt 设计主要遵循以下原则：
- 教学目标导向：围绕具体词汇、语法点、话题、场景或教学任务生成资源。
- 语言知识精准：保证词性、义项、搭配、语法点和语体要求准确。
- 教学适配：根据学习者水平、教学环节和使用场景控制难度。
- 结构化表达：使用清晰稳定的模板，便于复制、替换、存储和复核。
- 多模态协同：根据文本、图片、视频不同模态的特点设计提示词。
- 可迭代优化：通过“生成-筛选-反馈-修订”持续提高资源质量。

## 使用方法
1. 打开手册文件：[`《国际中文教学多模态资源生成Prompt手册》第三版.pdf`](./《国际中文教学多模态资源生成Prompt手册》第三版.pdf)
2. 明确教学任务和资源类型，如例句、图片、语法教学视频或数字人教学视频。
3. 准备必要输入信息，如目标词语、词性、释义、语法点、话题、场景、语体、学习者等级和教学目标。
4. 选择对应 Prompt 模板，替换模板中的变量。
5. 将 Prompt 输入大语言模型或多模态生成工具，获得初步资源。
6. 根据教学目标、语言准确性、文化恰当性和格式规范性进行人工复核与修改。
## 使用提醒
- 模型生成内容不宜直接作为最终教学资源，应经过人工审核。
- 文本资源应重点检查词义、语法、搭配和语用自然度。
- 图片资源应重点检查画面是否准确表现目标词义或场景。
- 视频资源应重点检查动作过程、镜头顺序、语义差异和口播节奏。
- 批量生成资源时，建议建立统一的命名、标注、筛选和校验流程。
